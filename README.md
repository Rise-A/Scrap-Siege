# Spire-Guard

This project was created in 2024 as my final project in a game design course. This repository was created to make my project's source files open to the public, rather than for version control. At this moment in time, I currently have no intentions of adding new content to this game.

## Technologies Used
* Unity 3D (Editor Version: 2022.3.5f1) - Game Engine 
* Blender - 3D asset creation, animation
* GIMP - Texture creation
